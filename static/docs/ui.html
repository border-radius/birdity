Интерфейс бирдости

Бирдость использует собственную библиотеку стилей для создания замечательных интерфейсов. В качестве препроцессора используется LESS (переменные, вложенные стили и другие восхитительные возможности).

Формы

Набор элементов скуден: только текстовое поле и кнопка. В конце концов, зачем кому-то может понадобиться что-то еще?

<input type="text" placeholder="Текстовое поле">
<button>Кнопка</button>

Зато, кнопка может иметь состояние .done

<button class="done">Готово!</button>

Лэндинг

Без посадочной страницы сегодня никуда, вот и мы туда же. Код лэндинга лаконичен и самочитаем: .background для фоновой картинки (или даже видео), .body для содержимого, которое можно отцентрировать через .center. И все это великолепие завернуто в .landing. Чума!

<div class="landing">
  <img src="http://i.imgur.com/XwwEFNy.jpg" class="background">
  <div class="body">
    <div class="center">
      <h1>Привет!</h1>
    </div>
  </div>
</div>

Привет!

Окно

Верстка окна очень проста: viewport по-вертикали делится на две области: 40% и 60%. Внутри меньшей располагается боковая панель, прилипая к правому краю области. Внутри большей области ‐ чат.

<div class="app">
   <div class="sidebar">
     <div class="content">
       Боковая панель
     </div>
   </div>
   <div class="chat">
     <div class="content">
       Чат
     </div>
   </div>
</div>

Добавьте контейнеру .app дополнительный класс .fullscreen чтобы размер чата совпадал с размером окна.

Карточка

Наверху боковой панели всегда находится карточка пользователя. В настоящий момент ее функционал минимален, но дальше всякое может произойти.

<div class="card">
  <img src="http://i.imgur.com/9KIYE30.jpg" alt="" class="userpic">
  <a href="" class="username">безумный пользователь</a>
  <button>Выйти</button>
</div>
безумный пользователь

Таб

Табы ‐ наше всё. Таб это ссылка на ленту, таб это пользователь в ростере, таб это чат в ростере. Табы вездесущи.

<div class="tab">
  <div class="user">
    <img src="http://i.imgur.com/9KIYE30.jpg" alt="" class="userpic">
    безумный пользователь
  </div>
  <div class="new">
    4
  </div>
</div>
безумный пользователь
4

В любой момент таб может оказаться активным ‐ .current. У активного таба счетчик непрочитанных сообщений заменяется на кнопку закрытия (кроме таба-ссылки).

безумный пользователь

Таб может быть ссылкой на ленту или черный список, если вместо .user использовать .link
